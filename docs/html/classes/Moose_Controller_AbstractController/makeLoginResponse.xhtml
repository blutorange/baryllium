Moose\Controller\AbstractController::makeLoginResponse

Called in the initial stage of the request processing when the user is not logged in and the page requires a login.

Redirects the user to the login page. After login, the user redirected to this page again. May be overridden for custom behaviour without calling the parent. It is currently overridden by the \Moose\Servlet\AbstractRestServlet.
